## üöÄ Project Overview
As automation accelerates, understanding job security becomes crucial. This project leverages **Random Forest Classification** and **Data Normalization** techniques to predict whether a specific job role falls into a Low, Medium, or High automation risk category based on current market trends.

### Key Features:
* **High-Precision Classification:** Achieved an **Accuracy of 91%** through optimized feature engineering.
* **Feature Importance Analysis:** Identifies key drivers such as AI Replacement Score and Skill Gap Index.
* **Scalable Architecture:** Integrated with a Streamlit front-end for real-time predictions.
* **Data-Driven Insights:** Analyzes workforce dynamics across 15,000+ records (2020-2026 simulation).
